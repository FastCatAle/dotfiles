# dotfiles for my Arch system

Currently WIP

## Programs used so far:  
- Text editor: vim
- Terminal: foot
- Screenshot utility: grim
- Audio server: pipewire
- Notification daemon: dunst
- Menu: wmenu

For neovim you need to install [Plug](https://github.com/junegunn/vim-plug)
